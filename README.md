# robotBox - 2D robot simulator
robotBox is a 2D robot simulator for Windows

## Features

* Single robot simulations
* Create walls
* Create paths
* Save and load maps
* Export simulations to video
* Multiple languages

## Supported robots

* ePuck
	* Proximity sensors
	* Floor sensors
	* Wheel motors
	* Leds
	* Wheel encoders

## System requirements

* .NET Framework 4.0 or higher
